## 总体
需要换行仅有：`题目结束后，每个选项，难易，答案`

## 题目
以数字开头，后面接`.、．`三种标点，并跟上题目内容，例如：
```
1.人类历史上以公有制为基础建立的法律是什么（   ）。
1、人类历史上以公有制为基础建立的法律是什么（   ）。
1．人类历史上以公有制为基础建立的法律是什么（   ）。
```
题目中间不能存在换行，例如：
```
1.人类历史上以公有
制为基础建立的法律是什么（   ）。
```

## 选项
以ABCDEFG开头，例如：
```
A 权利公平
B.机会公平
C．规则公平
D、救济公平  
```

## 难易与答案
难易可以不存在，但答案必需。以下都可：
```
难易程度：易     
答案：ABCD

【正确答案】C
【难易程度】易

【正确答案是】：A

【答案】：A
```
